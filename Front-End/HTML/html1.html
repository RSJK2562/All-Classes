Understand HTML Structure

What is HTML?

HTML stands for HyperText Markup Language. It is used to create web pages.

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
<title>My First Page</title>
</head>
<body>
<h1>Hello World</h1>
<p>Welcome to HTML Class 1</p>
</body>
</html>

Understanding Each Tag

  • <!DOCTYPE html> → Defines HTML5 document
  • <html> → Root of the webpage
  • <head> → Contains page information
  • <title> → Title shown in browser tab
  • <body> → Main visible content
  • <h1> → Main heading
  • <p> → Paragraph text
